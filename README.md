これは、デザインパターンの一つである「Visitor Pattern」を実装してみたサンプルプログラムです。
ロボットの各パーツを訪問先、各パーツに実行させるアクションを訪問者として「Visitor Pattern」を実装しています。
---
This is a sample program that implements the "Visitor Pattern", a design pattern.
The "Visitor Pattern" is implemented by treating each part of the robot as the place to be visited, and the actions to be performed by each part as visitors.
